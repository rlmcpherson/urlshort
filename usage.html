urlshort: url shortening api and service

POST to to /encode for a shortened url

to decode a shortened url without a redirect GET the shortened code at /decode/

example

  • encode url: 
        ❯ curl -X POST -d 'http://www.google.com' http://short.randallmcpherson.com/encode/     
              http://short.randallmcpherson.com/rDiNVbX
  • decode short url: 
          ❯ curl http://short.randallmcpherson.com/decode/rDiNVbX 
    
        http://www.google.com
  • follow short url: (can also use browser) 
          ❯ curl http://short.randallmcpherson.com/rDiNVbX 
    
        <a href="http://www.google.com">Moved Permanently</a>.
